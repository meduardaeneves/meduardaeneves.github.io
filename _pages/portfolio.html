GAMES

Some games, to be displayed in the website, needed to receive a code adjustment that was necessary to deploy them.

  • The website used to achieve this was the low-code Anvil Framework website;.
  • Despite the modification of the code on Anvil, each game page will provide a Github link to the raw code..

Some games created requires a GUI interface to work. Because of that, those games will not be able to be played on this website, but it will be provided and ".exe" to allow you to download and play.

  • The executable file was created with "Pyinstaller Library"
  • In some cases, your PC provides protection for the apps created like this. If that is the case, you need to exclude the file from the "Real-Time Scanning" section in your PCs firewall protection.
  • If you don't want to modify your PCs protection, you can access the github page with all the codes that were used to create the app, download them and then play them on your own code editor.
