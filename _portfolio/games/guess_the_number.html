This is a Rock, Paper, Scissors game you will play against the PC.

The game will happen to you choosing the element you want to enter::

  • Rock
  • Paper
  • Scissors

To play the game you can scroll down into the "Play Here" section or download all the python files in the game RESPOSITORY and execute it in your personal code editor.

Game Rules

  • The rules for the game are very simple:
    • Rock beats Scissors
    • Scissors beats Paper
    • Paper beater Rock
